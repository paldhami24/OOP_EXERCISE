# SET_08_FILE_IO

## Compilation Guide
To create all class files, run:
```bash
javac *.java
```

## Practical 26: Command-Line File Statistics
**Problem Statement:**
Write a program that will count the number of characters, words, and lines in a file. Words are separated by whitespace characters. The file name should be passed as a command-line argument.

**Sample Output:**
```text
> java practical26 sample.txt
Lines: 4
Words: 15
Characters: 82

```

## Practical 27: Student Records (File I/O)
**Problem Statement:**
Create a file named students.txt. Write at least three student records (roll number, name, marks) into the file. Read the file content and display all student records on the console. Handle exceptions like IOException properly using try-catch-finally.

**Sample Output:**
```text

Student Records:
101 Kesha 85
102 Smira 90
103 Ania 78

```

---

## Practical 28: Detailed File Analysis
**Problem Statement:**
Write a Java program that reads a text file named data.txt. The program should count and display the total number of lines, the total number of words, and the total number of characters (excluding spaces and newline characters). Handle FileNotFoundException and IOException.

**Sample Output:**
```text

Lines: 5
Words: 20
Characters (excluding spaces): 94

```
