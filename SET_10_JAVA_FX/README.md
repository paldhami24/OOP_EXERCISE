# SET_10_JAVA_FX

## Compilation Guide
To create all class files, run:
```bash
javac *.java
```

## Practical 33: Text Styling with VBox
**Problem Statement:**
Write a JavaFX program that displays five Text nodes vertically (stacked).
Use font Times New Roman, bold + italic, size 22 px.
Assign a random color and random opacity (0.3–1.0) to each text.
Center texts horizontally and add spacing.

## Practical 34: Registration Form with Validation
**Problem Statement:**
Create a registration form with fields:
Roll No (numeric)
Name
Age (numeric)
Email
Submit button
Requirements:
Validate inputs:
Roll No & Age → integers
Email → must contain @ and .
Show:
Success → confirmation alert
Failure → error alert

## Practical 35: Grade Distribution Bar Chart
**Problem Statement:**
Create a bar chart using Rectangle shapes to show grade distribution:
Projects → 20% (Red)
Quizzes → 10% (Blue)
Midterm → 30% (Green)
Final → 40% (Orange)
